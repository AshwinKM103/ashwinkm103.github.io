A serverless, multi-agent pipeline for automatic GitHub issue triage. Intake, Triage,
Assignment, and SLA agents coordinate over **AWS Lambda**, **EventBridge**, and **DynamoDB**,
with LLM-based classification using **Llama-3** and **Mistral-7B** via **AWS Bedrock**.

*Supervised by Prof. Yogesh Simmhan, IISc (Aug–Dec 2025). Code link to be added.*
